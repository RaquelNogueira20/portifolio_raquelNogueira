# 👟 Laboratório de Classificação Visual: Análise de Viés em IA

## 📝 Descrição do Projeto

[cite_start]Este projeto consiste em um experimento prático realizado no **Teachable Machine (Google)** para investigar como a seleção de dados influencia a lógica de algoritmos de aprendizado de máquina[cite: 1, 12]. [cite_start]O objetivo central é identificar o surgimento de vieses técnicos ao treinar um modelo de classificação de imagens utilizando categorias específicas[cite: 4, 5].

[cite_start]Desenvolvido como parte de uma **Atividade Formativa**, o sistema utiliza um dataset composto por fotos de 40 tênis diferentes, divididos entre as categorias "Original" e "Fake"[cite: 1, 2]. [cite_start]O experimento demonstra que critérios subjetivos na alimentação dos dados — como o estado de conservação do objeto — podem corromper a capacidade de predição do algoritmo[cite: 4, 5].

## 🚀 Tecnologias Utilizadas

* [cite_start]**Plataforma:** Teachable Machine (Google)[cite: 12].
* [cite_start]**Entrada de Dados:** Captura via Webcam[cite: 16].
* [cite_start]**Dataset:** 40 imagens de calçados categorizados entre originais e fakes.

## 📊 Memorial de Impacto e Ética

[cite_start]Com base no experimento técnico realizado, seguem as análises sobre o mecanismo e o impacto do viés[cite: 3]:

* [cite_start]**Mecanismo do Viés:** A seleção restrita de dados **corrompe** a lógica do algoritmo porque o sistema **interpreta** a sujeira ou a falta de um logo visível como evidência de falsificação[cite: 4, 5]. [cite_start]Dessa forma, o modelo **gera** uma visão distorcida da realidade ao focar em elementos estéticos temporários em vez de características estruturais do produto[cite: 4].
* [cite_start]**Consequência Social:** O sistema **marginaliza** o calçado e **despreza** o cuidado de quem o utiliza, criando uma classificação injusta baseada no desgaste[cite: 6, 7]. [cite_start]Essa falha **invisibiliza** a autenticidade do item e **promove** um julgamento equivocado sobre o indivíduo que possui o objeto mais gasto[cite: 6, 7].
* [cite_start]**Ação Mitigadora:** O ser humano **faz** uma curadoria detalhada por modelos de cada marca para estabelecer critérios técnicos de originalidade[cite: 8]. [cite_start]Essa intervenção **garante** a equidade no processo, pois o especialista **define** com exatidão o que constitui um item original, independentemente do seu estado de limpeza[cite: 8].
